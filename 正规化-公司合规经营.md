# 正规化 — 公司合规经营

> 从「正规化-总结汇总.md」拆分，聚焦公司财务隔离与合规经营

***

## 一、背景

父亲（李学起）注册的一人有限公司（100%持股）存在两个核心风险：

- **人格混同风险**：个人账户和公司账户混用，公司欠款可能拿个人财产承担，个人欠债也可能拿公司财产承担
- **税务合规风险**：公司进项发票不合规（农产品收购发票，行业典型稽查高风险），持续经营意味着随时可能被查补税（13%增值税+滞纳金+罚款）

**已决策：注销旧公司、注册新公司作为风险隔离手段。**

**新公司安排**：

- 初期由母亲作为独资股东注册新的一人有限公司，并担任法定代表人，与父亲名下的旧公司做切割
- 后续根据需要转变为多股东结构，法定代表人也可能调整
- 定位：承接木材厂经营业务，并据此安排父亲、母亲及其他家人的工资与分红

**律师已确认的要点**：

- 注销旧公司需经过清算，涉及债务继承，流程相对复杂但仍为可行选项
- 新公司从注册之日起即做好财务隔离，不再公账私账混同
- 父亲保持名下无财产的状态，新公司主体放在母亲名下，进一步隔离父亲个人风险

***

## 二、**注销旧公司、注册新公司双线**

**A. 旧公司注销流程**（封顶风险）

| 步骤     | 关键动作                                                                           |
| ------ | ------------------------------------------------------------------------------ |
| 债权债务清理 | 清理应收账款（67.8万）、应付账款、欠税等，制作清算报告。与代账公司拉取最新科目余额表确认各项余额                             |
| 应收账款处理 | 决定67.8万归属：注销前收回 / 债权转让给新公司 / 父亲个人承接。逐笔确认客户签字单据，无据则补签对账单或合同。律师提醒：私下账本对方不认则无法律效力 |
| 税务清算   | 确认进项不合规的补税金额与滞纳金，结清欠税、补申报、办理清税证明，这是注销难点                                        |
| 注销登记   | 工商注销登记、银行账户销户、印章销毁                                                             |
| 资产迁移   | 与律师/会计确认：设备、原木库存等如何合规迁移到新公司                                                    |

**B. 新公司注册筹备**（承接业务）

| 步骤         | 关键动作                                                                                                                                      |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| 主体设计       | 母亲独资设立一人有限公司，担任法定代表人；预留后续转多股东的股权结构调整空间（律师确认最佳时点和方式）                                                                                       |
| 税务身份评估     | 起步是否可先用小规模纳税人过渡（取决于客户是否接受3%专票）；若为一般纳税人，增值税按13%计算（销项-进项），**多拿进项专票**是关键。税负通过定价转嫁客户，实际非全由公司承担                                                |
| 银行开户       | 新公司对公账户开立，从注册之日起所有经营收支走对公                                                                                                                 |
| 建账起步       | 代账公司（广聚会计服务）协助：账套搭建、税控盘申领、发票领用全套                                                                                                          |
| 财务隔离（3条原则） | ①**收款**：所有客户付款只转新公司对公，父母微信不收客户钱，父亲不代收；②**付款**：采购开票走对公，工资对公发放+工资条，日常开支能开票走对公、未开票保留单据；③**个人开销**：父母生活费走"工资"或"分红"形式，父亲虽不持股不担任法人，可按"劳务"或"工资"取酬 |
| 客户过渡       | 与旧客户重新签订购销合同，明确付款走新公司对公账户（律师确认）                                                                                                           |

> **沟通注意事项**：与代账公司/律师沟通时，不提"贷款纠纷""资产保护"等敏感词，只说"规范经营""合规要求"。如会计说做不了则记录，可能需换代账公司。让会计出书面【旧公司注销+新公司建账】过渡方案。
